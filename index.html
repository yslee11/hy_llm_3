보행환경 인식 조사

안녕하세요. 본 설문은 한양대학교 도시공학과 도시설계 및 공간분석 연구실에서 서울시 보행환경에 대한 가로 보행 환경 인식을 평가하기 위해 진행됩니다.

평가 방법

  • ● 제시되는 사진을 보고 "이 장소에서 보행할 때 느끼는 보행환경 수준"을 1-5점으로 평가합니다.
  • ● 1점 = 매우 그렇지 않다, 5점 = 매우 그렇다
  • ● 고려 대상: 보행자 시설, 보행 공간, 차량 통행량, 녹지 등 보행 관련 요인
  • ○ ※ 탈것에 대한 인식은 평가에서 제외해주세요.

평가 항목

  • ① 안전함 (Safety) : 해당 장소가 얼마나 안전해 보이는가?
  • ② 활기참 (Lively) : 해당 장소가 얼마나 활기차고 생동감 있는가?
  • ③ 아름다움 (Beauty) : 해당 장소가 미관상 얼마나 아름다운가?
  • ④ 부유함 (Wealthy) : 해당 장소의 관리가 잘 되어 있다고 느껴지는가?
  • ⑤ 지루함 (Boring) : 해당 장소가 얼마나 단조롭고 지루한가?
  • ⑥ 우울함 (Depression) : 해당 장소가 얼마나 우울하고 삭막한가?

기본 정보

이미지를 보고 보행환경 수준을 평가해주세요

1 = 매우 그렇지 않다 / 5 = 매우 그렇다

③ 아름다움(Beauty): 해당 가로경관은 미관상 아름다워보이시나요?

④ 부유함(Wealthy): 해당 가로경관은 관리가 잘 되어 있다고 느껴지나요?

⑤ 지루함(Boring): 해당 가로경관은 단조롭고 지루하게 보이나요?

⑥ 우울함(Depression): 해당 가로경관은 우울하고 삭막해보이시나요?
